# Node.js Basic for routing

This is a simple project to show basic html page and routing using node.js

## Requirements
* Node.js
* Any web browser


### 1. how to install dependencies
```sh
npm i
```

### 2. how to view pages
```sh
http://localhost:3000
```
```sh
http://localhost:3000/about
```
```sh
http://localhost:3000/contact
```
